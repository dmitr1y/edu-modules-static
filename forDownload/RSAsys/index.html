Генерация больших простых чисел

    Программа предназначена для визуализации протоколов, основанных на криптосистемах с открытой частью ключа. В программе представлены стандартные протоколы, обсужденные на лекциях.

    Системные требования

    • Операционная система: Windows
    • Необходимые пакеты: Java SE Runtime Environment

    Инструкции

    Работа будет происходить внутри локальной сети, и запускать программы будем на одном компьютере. Также будем использовать сниффер, для перехвата сообщений между избирателями, избирателями и регистратором, избирателями и счетчиком.

    1. Настроим регистратор
    2. Добавим 4 избирателя и 4 кандидата для голосования
    3. Настроим сервер регистратора
    4. Запустим счетчик и настроим счетчик
    5. Запускаем клиент для первого избирателя и подключаемся к серверу регистратора
    6. Вводим имя первого избирателя
    7. Выбираем, за кого голосует первый избиратель (кандидат No4)
    8. Запускаем клиент для второго избирателя, подключаемся к серверу регистратора
    9. Выбираем, за кого голосует второй избиратель (кандидат No1)
    10. Также запускаем клиенты для третьего и четвертого клиента
    11. Голосуем 3-им клиентом за 3-его кандидата
    12. Голосуем 4-ым клиентом за 4-ого кандидата
    13. Счетчик получает кусочки голосов, подсчитывает, выводит итог и отсылает клиентам
    14. Каждый избиратель получает итог голосования
    15. Теперь посмотрим, что смог отловить сниффер, при подключении клиента к регистратору Мы видим, что клиент отсылает запрос серверу на получение ключа по открытому каналу
    16. Сервер регистратора передает ключ клиенту по отрытому каналу
    17. Клиент передает свои личные данные регистратору в зашифрованном виде
    18. Один избиратель передает другому избирателю кусочек своего голоса в зашифрованном виде

    19. Клиент передает счетчику несколько зашифрованных кусочков голосов (Скриншот отсутствует). Перехваченная строка выглядит следующим образом:
