Программа для выражения конъюнкции через конечный набор булевых функций

    В программе реализован оптимизированный алгоритм выражения конъюнкции, то есть в более кратчайшем виде, чем если бы это делать вручную (Иногда ответы совпадают. Поэтому выражение в более кратчайшем виде по большей части применимо к функциям от большого числа переменных (7, 8 и так далее)), так как для такой функции выражение конъюнкции в кратчайшем виде вручную занимает много времени.

    Системные требования

    • Операционная система: Windows
    • Необходимые пакеты: Microsoft .NET Framework 4

    Инструкции

    • Скачать модуль
    • Разархивировать архив "Function.zip"
    • Открыть файл Function.exe
