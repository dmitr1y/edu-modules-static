Алгоритм возведения числа в многоступенчатую степень по модулю

    описание

    Системные требования

    • Операционная система: Windows
    • Необходимые пакеты: Java SE Runtime Environment

    Инструкции

    В папке «TestMath» содержится исполняемый файл «Math.exe». Чтобы протестировать работу программы на Windows Вам необходимо запустить именно его и следовать инструкциям на экране.
    Если же используете *nix систему, то программу можно запустить из терминала используя команду «python3 /…/MathSource.py».
    При тестировании советуется не вводить больше более чем четырехзначные числа, во избежание долгих подсчетом или вылета программы из-за недостатка оперативной памяти. В основной директории есть файл «MathSource.py», в котором содержится исходный код программы. Вы можете открыть его и просмотреть в любом текстовом редакторе типа «Блокнот». Основная функция «main», в которой реализована основная задача экзамена прокомментирована в псевдокоде с пояснениями. Программа написана на интерпретируемом языке программирования Python версии 3. Именно этот язык был выбран не случайно, по моему мнению, он является наилучшим архитектурным решением в данной ситуации, так как обеспечивает быструю работу с длинной арифметикой и содержит большое количество методов работы с различными типами коллекций
