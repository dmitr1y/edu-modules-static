Биометрическая идентификация личности по голосу

    В проекте рассматривается решение задачи биометрической идентификации личности по голосу. Решение включает в себя разработку и программную реализацию в среде MATLAB трех алгоритмов: аутентификации человека по голосу, идентификации пола человека по голосу и идентификации эмоционального состояния человека по голосу. Проверена принципиальная возможность использования для решения этих задач комбинации алгоритма DTW (Dynamic Time Warping) с быстрым преобразованием Фурье. Для тестирования алгоритмов использованы 27 файлов с аудиозаписями (*.wav), взятыми из берлинской базы данных эмоциональной речи (Emo-DB). Данная база данных была разработана в техническом университете Берлина в рамках гранта № SE 462/3-1 to W. Sendlmeier, финансируемого Немецким научно-исследовательским обществом (Deutsche Forschungsgemeinschaft - DFG). Исследователи в области обработки аудио-информации в разных странах мира часто используют ее при разработке систем распознавания эмоционального состояния.

    Системные требования

    • Операционная система: Windows
    • Необходимые пакет: MatLab

    Инструкции

    Программное обеспечение можно условно разбить на три части:

    1. Четыре базовых функции, разработанные в проекте:
      • WhoIsIt - программная реализация алгоритма аутентификации человека по голосовому паролю
      • SexOfPerson - программная реализация алгоритма идентификации пола человека по голосу
      • HowAreYou - программная реализация алгоритма идентификации эмоционального состояния человека по голосу
      • Spectrum - вспомогательная функция, предназначенная для расчета спектра на основе стандартной Matlab-функции, реализующей БПФ, и его визуализации.
    2. Одна свободнораспространяемая функция visualDTW, реализующия алгоритм DTW и имеющая возможность графической визуализации его работы.
      Функция была загружена с сайта фирмы MathWorks, производителя Matlab.
    3. 27 файлов с расширением *.wav - аудиозаписи из Берлинской базы эмоциональной речи, используемые в проекте для тестирования разработанных алгоритмов биометрической идентификации личности по голосу.
      Для работы с программным обеспечением необходимо:
      1. Распаковать архив в любую папку
      2. Запустить программу MATLAB
      3. В поле со списком «Current Folder» указать или выбрать путь к папке, в которой находятся распакованные файлы.
      4. В командном окне Command Window набрать имя программы, реализующий нужный алгоритм, и нажать клавишу «Enter», т.е. последовательно выполнить функции:
        • WhoIsIt
        • SexOfPerson
        • HowAreYou

    Все функции после своего завершения оставляют на экране ВСЕ ГРАФИКИ, сгенерированные в ходе работы и иллюстрирующие работу разработанных алгоритмов.
    После анализа этих графиков, окна, их содержащие, могут быть закрыты вручную.
