Программа “Клеточный автомат”

    Программа “Клеточный автомат” предназначена для демонстрации работы клеточных автоматов, где вы сами можете создавать правила, добавлять новые типы клеток, и задавая начальную конфигурацию наблюдать за работой алгоритма

    Системные требования

    • Операционная система: Windows
    • Необходимые пакеты: Java SE Runtime Environment

    Инструкции

    • Скачать модуль
    • Открыть файл cellular__automat.jar
