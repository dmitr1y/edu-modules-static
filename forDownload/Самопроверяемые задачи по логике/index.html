Самопроверяемые задачи по логике

    Полная инстуркция по работе с программой находится внутри программы. Модуль служит для генерации задач с логическими высказываниями, генерации соотвествующих интерпретаций на человеческом языке и проверки задач. Проверка задач происходит поэтапно по правилам силлогизмов: Modus ponens, modus tollens, дизьюнктивный силлогизм, гипотетический силлогизм, разделительный силлогизм, простая конструктивная дилемма, сложная конструктивная дилемма, простая диструктивная дилемма, сложная диструктивная дилемма.

    Системные требования

    • Операционная система: Windows, Linux, Mac
    • Необходимые пакеты: Java SE Runtime Environment (Windows)

    Инструкции

    • Скачать модуль
    • Разархивировать архив "Исполняемые файлы + задача.zip"
    • Открыть файл ProtocolVisualizer.jar
