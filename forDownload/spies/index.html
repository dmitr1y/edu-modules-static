Шпионы

    Программа предназначена для визуализации протоколов, основанных на криптосистемах с открытой частью ключа. В программе представлены стандартные протоколы, обсужденные на лекциях.

    Системные требования

    • Операционная система: Windows

    Инструкции

    У N шпионов есть числа, являющиеся частями ключа, который является упорядоченным набором чисел: у №1 - первый элемент набора, у 2-го - второй и т.д. Этим шпионам можно задавать вопросы типа "Вы уже знаете ключ?" или "Знает ли №k ключ?", ответ на которые может быть только "да" или "нет".

    На входе программы имеется набор ключей, среди которых есть искомый ключ. Ваша программа должна задавать вопросы шпионам (определяя тип вопроса и кому он задается) до тех пор, пока кто-то из шпионов не сможет назвать ключ.


    После запуска программы пользователь должен ввести в поле ввода в правой части программы набор из ключей. В одной строке должен быть только один ключ. Ключ должен состоять из натуральных чисел, разделенных пробелами. Числа меньшие или равные нулю и другие символы будут проигнорированы. Ключи должны быть одинаковой длины. Ключи, длина которых меньше, считаны не будут. Лишние части в ключе, длина которого больше, будут проигнорированы. Пример верно введенных данных можно увидеть в поле ввода. ВНИМАНИЕ: верный ключ указывается в первой строке.
