Применение нечёткой логики для анализа результатов тестирования

    Основной задачей проекта было предсказание результата решения студентом задачи на основе имеющихся данных о решении этой задачи другими студентами и его решениях других задач.

    Системные требования

    • Операционная система: Windows
    • Необходимые пакеты: Окружение языка R, Распространяемый пакет Visual C++ для Visual Studio 2015

    Инструкции

    Для того чтобы успешно запустить программу, необходимо скачать архив, распаковать его и запустить файл Neuro.exe, дальнейшие инструкции будут получены в ходе работы с программой. first.csv - тестовые данные для обучений нейросети; second.csv - тестовые данные для предсказания. Если файл находится в директории с программой, то можно указать только название файла.

    • Скачать модуль
    • Разархивировать архив "Pre-Release.zip"
    • Установить необходимые пакеты. Пакет языка R установить в папку с программой. Установка пакета должна быть выполнена в папку с именем "RI
    • Открыть файл Neuro.exe от имени администратора
