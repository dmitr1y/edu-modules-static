Программа "Регулярные выражения"

    Программа "Регулярные выражения" предназначена для автоматической проверки грамматик. Программа включает в себя также набор заданий и редактор. Редактор позволяет создавать новые задачи. Рекомендуется созданные задачи сохранять в папку Asserts.

    Системные требования

    • Операционная система: Windows
    • Необходимые пакеты: Net Framework

    Инструкции

    • Скачать модуль
    • Разархивировать архив "Регулярные выражения.zip"
    • Открыть файл "Регулярные выражения.exe"
